Call for Photos

A print version of the guidebook is currently in deveolpment (its a long way out, but you can see the WIP version here). This version will be similar to the regular pdf except it will feature additional action and scenery photos. If you have photos that you want to be considered for the guidebook get in touch at an.child@gmail.com. Anyone whose photo gets included in the guidebook will get a credit in the index and a free copy of the print guide. If you send a photo with a person in it please get their premission before sharing.
