The Garden

A collaborative guidebook for the Garden climbing area generated using the LocalBoulders framework. This book is a living document which is continuously being updated. Follow the link below to download the latest version.

Download the latest version as a PDF

About the Project

Scope

This guidebook is intended to be a comprehensive guide to the Garden climbing area located outside of Sweet Home, Oregon. Additional sections are included to document other nearby areas, but these areas are not covered in as great of detail. A PDF version is available for free online and a print version is in development. The guidebook is a living document that has been continuously developed since July of 2022.

Sharing

This website, the Garden Guidebook, and the python code used to generate it are all being hosted on GitHub as open source software. That means that you are free to use, copy, share, and modify it as you please. That doesn't mean that I won't be upset if you just post the whole thing on Mountain Project verbatim.

Information Sources

Much of the information in this book has been sourced from own my first hand knowledge of the area. I have been climbing at the Garden for over a decade and have climbed or attempted roughly 75% of the routes documented in this book. I have also actively collaborated with many other Garden regulars in the creation of this document.

Several other DIY guides to the area have been authored over the years. At many times these texts along with online resources like Mountain Project have been used as references for names and locations, but the descriptions and topos in this guidebook are all original work.

Ways to Contribute

Send Your Feedback

Is there any information in the book that you find confusing? Are there any route names or grades that you disagree with? Is your buddy's sick FA not credited in the index? If you have any thoughts on how the information in the guidebook can be expanded or improved please get in touch with me at an.child@gmail.com.

Call for Photos

A print version of the guidebook is currently in development. This version will be similar to the regular pdf except it will feature additional action and scenery photos. If you have photos that you want to be considered for the guidebook get in touch at an.child@gmail.com. Anyone whose photo gets included in the guidebook will get a credit in the index and a free copy of the print guide. If you send a photo with a person in it please get their permission before sharing.

Submit Change Requests Through GitHub

If you are familiar with Git you are welcome to collaborate and submit proposed changes directly on the platform.
